# WordPress coding standards

## Installation
Install these composer packages just like this repository is doing:
- [PHP_CodeSniffer](https://github.com/squizlabs/PHP_CodeSniffer)
- [WordPress Coding Standards for PHP_CodeSniffer](https://github.com/WordPress/WordPress-Coding-Standards) 

## Visual Studio Code

* Install [PHP Sniffer & Beautifier for VS Code](https://github.com/valeryan/vscode-phpsab)
* Install [vscode-phpcs](https://github.com/ikappas/vscode-phpcs.git)
